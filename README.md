# Nabatian Restaurant - Demo Websites

Zwei moderne Website-Versionen für Nabatian Restaurant & Catering am Sonnenbrunnen, Heilbronn.

## 🎯 Versionen

### Design 2 (Astro - SEO Beast)
**Pfad:** `design2/`
**Tech:** Astro 6.0.8 + React Islands + Tailwind CSS
**Status:** ✅ Production-Ready (7 Seiten komplett)
**Deployment:** Vercel (Static Site)

**Live Demo:** [https://nabatian-design2.vercel.app](https://nabatian-design2.vercel.app)

**Features:**
- 7 vollständige Seiten (Homepage, Restaurant, Catering, Kontakt, Galerie, Impressum, Datenschutz)
- SEO-optimiert (sitemap.xml, robots.txt, Schema.org)
- Burgundy Branding (#711521)
- Sanity CMS vorbereitet
- Mobile-responsive

---

### Design 3 (Next.js Premium - 3D Experience)
**Pfad:** `design3/`
**Tech:** Next.js 16.2.1 + React Three Fiber + Framer Motion + Lenis
**Status:** ⚠️ Homepage fertig, weitere Seiten in Entwicklung
**Deployment:** Vercel (SSG)

**Live Demo:** [https://nabatian-design3.vercel.app](https://nabatian-design3.vercel.app)

**Features:**
- 3D Hero mit rotierendem Logo (React Three Fiber)
- Smooth Scroll (Lenis)
- Advanced Animations (Framer Motion)
- 3D-Tilt Hover Cards
- Gleiche Burgundy-Branding wie Design 2

---

## 🚀 Lokale Development

### Design 2 (Astro)
```bash
cd design2
npm install
npm run dev
# → http://localhost:4321
```

### Design 3 (Next.js)
```bash
cd design3
npm install
npm run dev
# → http://localhost:3000
```

## 📦 Production Build

### Design 2

```bash
# Output: dist/
```

