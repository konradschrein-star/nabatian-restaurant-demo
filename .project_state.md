## 2. Architecture: "The 1:1 Astro Port" Letzter Stand
Um die kompletten UI-Animationen, Elementor-Skripte und das Dropdown-Verhalten **exakt 1:1** zu bewahren, wurde die Roh-HTML aus der alten Website in Astro portiert anstatt sie via Tailwind neu zu bauen. 
- **Astro als Wrapper:** Das gesamte HTML der originalen Website liegt in `src/pages/index.astro`. Astro dient hier als Build-Tools.
- **Assets:** Alle originalen CSS, JS und Bilder (aus `Persisches Restaurant _ Nabatian_files`) wurden direkt nach `public/assets/` migriert. Astro serviert sie von dort perfekt.
- **Scripte:** Alle `<script>`-Tags via `is:inline` markiert, damit der Astro-Bundler sie unangetastet lässt.
- **Webfonts:** FontAwesome wurde auf ein funktionierendes CDN geändert, um kaputte Icons zu unterbinden.
- **Data-Layer:** In `src/data/menu.ts` liegt statisch schon mal das Framework für die Speisekarte, um das HTML-Menü demnächst durch Schleifen dynamisch zu machen.
